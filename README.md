# Syntagmax VS Code Extension

VS Code support for the Syntagmax DSL, providing syntax highlighting and language intelligence.

## Features

- **Syntax Highlighting**: Robust highlighting for artifacts, attributes, and types.
- **Linting**: On-the-fly validation of your `.stmx` and `.syntagmax` files.
- **Completions**: Smart suggestions for keywords and built-in types.

## Prerequisites

- **Python 3.x**: Required to run the Language Server.
- **Dependencies**:
  ```bash
  pip install pygls lark
  ```

## Installation

1. Clone the repository.
2. Open the `syntagmax-vscode` folder in VS Code.
3. Run `npm install` and `npm run compile`.
4. Press `F5` to launch the extension in a new window.
